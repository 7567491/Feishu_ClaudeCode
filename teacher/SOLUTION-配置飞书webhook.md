```

https://open.feishu.cn/app/cli_a9ad59fd26389cee
```

### 步骤3：配置事件订阅

1. **进入菜单**: 事件与回调 → 事件配置

2. **配置请求地址URL**:
   ```
   http://139.162.52.158:33301/webhook/feishu
   ```

3. **点击验证按钮**
   - 系统会自动发送验证请求
   - 我们的服务会自动响应（已测试通过）

4. **添加事件订阅**:
   - 点击"添加事件"
   - 搜索并选择: `接收消息 v1.0` (im.message.receive_v1)
   - 保存配置

5. **确认配置生效**
   - 状态应显示"已启用"

### 步骤4：确认机器人在群组中

1. 在"AI之初"群聊中检查成员列表
2. 如果AI初老师不在群组中，@AI初老师 邀请加入

在群聊中发送:
```

```
